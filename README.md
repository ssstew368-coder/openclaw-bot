# Soré — Agent IA Personnel de Steven

Agent IA personnel basé sur OpenClaw, connecté à Telegram (@SungSore_Bot).

## Stack
- OpenClaw (agent framework)
- Gemini 2.0 Flash (tâches générales)
- Groq Llama 3.3 70B (coding)
- Telegram (interface)
- Render (hébergement H24)

## Variables d'environnement requises
- `TELEGRAM_BOT_TOKEN` — Token du bot @SungSore_Bot
- `GEMINI_API_KEY` — Clé API Google Gemini
- `GROQ_API_KEY` — Clé API Groq
